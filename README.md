## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo

```sh
https://github.com/NakulPrasad/Data-Analysis-Task.git
```
2. Install NPM packages

```sh
yarn install
```

5. RUN

```
yarn run dev
```
